# Pokepay Partner SDK for C#

### Setup

* Install .NET 5.0 or Later from the [Download site](https://dotnet.microsoft.com/learn/dotnet/hello-world-tutorial/install)

### Build
```
$ make build
```

### Run test
```
$ make test
```
